# centrifugal-pump-cavitation-data
This repository contains the supplementary experimental data, signal processing outputs, and visual results for my Master's thesis on modeling cavitation phenomena in a centrifugal pump.

Repository Structure
The dataset is organized into the following categories:

Pump Operational Data (10-04-26_Measurements_Pump_Data): Contains the operational parameters logged during testing (e.g., RPM, flow rate, pressures, temperatures) and their corresponding time-domain plots.

Vibration & Acoustic Signals (Accelerometers Signals, Microphone Signals, 10-04-26_Accelerometers_Microphone_Measurements): Raw and processed time-domain and frequency-domain acoustic and vibration measurements captured from the pump casing and surrounding environment.

Acoustic Camera Analysis (Acoustic Camera Measurements, Acoustic_Camera_FFTs, Processed_Acoustic_Camera_FFTs): Raw spatial acoustic data, alongside the Fast Fourier Transform (FFT) frequency-domain plots used to isolate and analyze cavitation signatures.

Visual Footage (High-Speed Camera PNG): Extracted image frames from the high-speed camera used to visually capture, observe, and validate the cavitation bubble dynamics.
